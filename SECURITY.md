# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing `contact@md4.tech`.

**Please do not report security vulnerabilities through public GitHub issues.**

We will respond within 48 hours with next steps. We appreciate your responsible disclosure and will keep you informed throughout the process.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Update Process

1. Vulnerability is reported and confirmed
2. Fix is developed in a private repository
3. Security advisory is published
4. Patch is released

Thank you for helping keep this project secure.
