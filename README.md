# Settings things up

Here what you should know about using PTAH.

## Use PTAH

Make sure you have [pnpm](https://pnpm.io/fr/installation) installed.

Then, please run:

```bash
pnpm i
pnpm build
pnpm start
```

## Develop PTAH

To contribute to PTAH, please run:

```bash
pnpm dev
```
